## Editing Notes

1. **Pacing:** Fast cuts (2-4 seconds per shot), don't let anything linger
2. **Screen recordings:** Zoom into relevant areas, don't show full desktop
3. **Transitions:** Clean cuts or subtle zooms, no cheesy effects
4. **Color:** Dark theme, green accents matching AgentWire brand
5. **Sound design:** Subtle UI sounds for button presses, notifications
